D — Select → Mini Toolbar → Translate

Select any text and a floating mini toolbar appears (like Notion/Medium). Click "Translate" to see the result in a popup. Clean, discoverable, non-intrusive. Also works for double-click on single word.

Recommendation

D (Mini Toolbar) — most polished and discoverable. Works for both single words and phrases. The toolbar can also host Copy and Bookmark, consolidating actions. Falls back gracefully if translation is disabled.
