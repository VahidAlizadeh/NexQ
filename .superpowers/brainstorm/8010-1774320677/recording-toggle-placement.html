A: Dedicated Row with Red Accent

Recording toggle gets its own bordered row between Scenario and Remember checkbox. Red dot + subtle red border make it visually distinct as a "recording" control without being loud. Toggle defaults to last-used state.

B: Compact Inline with Remember Row

Recording toggle shares the row with the "Remember settings" checkbox. Compact — no extra vertical space. "REC" label + red dot keeps it identifiable. Trade-off: less visible, might be missed.

Compact View (remembered setup)

In both options, when the user has saved preferences, the compact view shows a REC badge alongside the Audio Mode and Scenario badges. One-click start includes recording state.
